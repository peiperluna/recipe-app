🍽️ 오늘 뭐먹지? (Recipe Finder App)

🚀 프로젝트 소개
한국 공공 API를 활용한 레시피 검색 및 추천 웹 애플리케이션

✨ 주요 기능
ㆍ 🔍 레시피 검색
ㆍ 🔥 인기 레시피 캐러셀
ㆍ 📋 레시피 상세 정보 제공

🛠 기술 스택
ㆍ React
ㆍ React Router
ㆍ Public API (식품안전나라)

📦 주요 컴포넌트
ㆍ SearchBar
ㆍ RecipeList
ㆍ RecipeCard
ㆍ RecipeDetails
ㆍ Carousel

🌟 핵심 특징
ㆍ 반응형 디자인
ㆍ 현대적인 UI/UX
ㆍ 간편한 레시피 검색

🔗 API 정보
ㆍ 식품의약품안전처 공공데이터 활용

📌 향후 계획
ㆍ 북마크 기능
ㆍ 레시피 필터링
ㆍ 사용자 레시피 추가 기능
